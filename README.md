# mamba

TUI framework (WIP)

nothing is set in stone, expect braking changes

## outline

- every component is a frame component
- frame controls everything related to design
  - paddings
  - margins
  - colors
  - borders
  - border color
- main window is still a frame
- no keybinds by default
- assignable actions
- programmed navigation

components

- tabs
- splits
- loading bar
- input
- text
